# Installation

## Stable release

To install Care Plugin, run this command in your terminal:

```sh
pip install Care Plugin
```

## From source

The source files for Care Plugin can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_plugin).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_plugin
```

Or download the [tarball](https://github.com/ohcnetwork/care_plugin/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_plugin/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_plugin
pip install .
```
